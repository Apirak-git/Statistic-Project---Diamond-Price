# Statistic-Project---Diamond-Price
รวมข้อมูล เนื้อหา ของโปรเจกต์รายวิชาสถิติ เรื่อง ราคาเพชร

# รายละเอียดของข้อมูลที่ได้จากไฟล์ .csv

## ความหมายในแต่ละคอลัมน์
- X : ลำดับที่ของข้อมูล
- carat : น้ำหนักเพชร
- cut : คุณภาพการเจียระไน
- color : ระดับความเข้มสีของเพชร
- clarity : ความสะอาด ตำหนิ
- depth : เปอร์เซนต์ความลึกของเพชร
- table : ความกว้างของหน้าเพชร
- price : ราคาเพชร มีหน่วยคือ USD
- x, y, z : ขนาดมิติของเพชร กว้าง ยาว สูง หน่วยเป็น มิลลิเมตร
